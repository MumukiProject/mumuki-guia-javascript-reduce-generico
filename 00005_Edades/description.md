Queremos calcular la suma de todas las edades que están en una lista de personas. Por ejemplo:

```js
let personas = [
    {
      nombre: "Gra", 
      edad: 34
    }, 
    {
      nombre: "Ana", 
      edad: 19
    },
    {
      nombre: "Tito", 
      edad: 6
    }
  ];


ム sumaDeEdades(personas);
59
```

> Definí la función `sumaDeEdades` utilizando `reduce`.
